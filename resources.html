Useful Resources & tips for front end web development

Listed here are some useful Resources & tips to help you become a great front end web developer

Tips for maximising SEO

  • Remember good code = good SEO. Use semantic HTML5 tags to structure your content
  • Mention keywords throughout your site. Check out https://www.wordtracker.com/ to find keywords
  • Write in understandable human language
  • Link to other pages within your site.
  • Make sure your webpages load fast ie optimise your images
  • Use keywords in your images alt attributes.
  • If you can keep your domain name as long as you can. In this case your domain name age matters

Tips for Maximising accessibility

  • Proper use of semantic HTML5 tags to structure your content is so important for accessibility
  • Include alt text for images where needed. If the image is just decorative leave the alt attribute empty
  • Consider your use of colour and contrast. You can try out various colour blindness filters on some websites. coolor.co has a great one
  • Give your links descriptive names. Avoid saying ‘click here’ and use ‘Contact Information’ instead
  • Use label tag for your forms
  • Never use tables for layout and use for tabular data when needed
  • Make dynamic content accessible
  • Use the WAVE browser extension to check your website for accessibility

Tips for Clean & Concise Code

  • Indent your code properly
  • Use descriptive ids and class names
  • Don’t overuse classes and ids
  • Use all lowercase for HTML and CSS / camelCase for javascript
  • Comment your code with universal easy to understand words
  • Put your code in logical order starting from top to bottom
